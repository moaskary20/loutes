## لوحة تحكم Loutes Store (Laravel + Filament)

هذا المشروع عبارة عن لوحة تحكم لمتجر **Loutes Store** مبنية باستخدام **Laravel** و **Filament** مع دعم كامل للغة العربية.

### المتطلبات
- **PHP 8.2+**
- **Composer**
- خادم قاعدة بيانات (MySQL أو SQLite أو غيره)

### خطوات تشغيل المشروع لأول مرة

1. **تثبيت الاعتمادات**

```bash
composer install
```

2. **إعداد ملف البيئة `.env`**
- تأكد أن ملف `.env` موجود (تم إنشاؤه تلقائياً من `.env.example` عند إنشاء المشروع).
- عدّل إعدادات قاعدة البيانات حسب بيئتك.
- تم ضبط إعدادات اللغة في `config/app.php` لتكون:
  - `locale` = `ar`
  - `fallback_locale` = `ar`

3. **توليد مفتاح التطبيق (إن لم يكن موجوداً)**

```bash
php artisan key:generate
```

4. **تشغيل الترحيلات وإنشاء مستخدم المدير**

```bash
php artisan migrate
php artisan db:seed --class=AdminUserSeeder
```

سيتم إنشاء مستخدم أدمن افتراضي بالبيانات:
- **البريد:** `admin@loutes.test`
- **كلمة المرور:** `password`

5. **تشغيل الخادم**

```bash
php artisan serve
```

ثم افتح المتصفح على:

```text
http://127.0.0.1:8000/admin
```

وسجّل الدخول ببيانات الأدمن أعلاه.

### الترجمة العربية

- تمت إضافة ملفات الترجمة العربية في مجلد `lang/ar`:
  - `auth.php`
  - `pagination.php`
  - `passwords.php`
  - `validation.php`
- تم ضبط تطبيق Laravel بالكامل لاستخدام العربية، وستظهر رسائل التحقق وتسجيل الدخول بالعربية.

### ملاحظات
- يمكنك تغيير بريد/كلمة مرور الأدمن من لوحة التحكم بعد تسجيل الدخول أو مباشرة من قاعدة البيانات.
- لاحقاً يمكننا إضافة حزم ترجمة خاصة بـ **Filament** لجعل كل عناصر الواجهة (القوائم، الأزرار، الرسائل) بالعربية بالكامل.
